# 🌸 貸款計算器 | Sakura Loan Calculator

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
![Version](https://img.shields.io/badge/Version-1.0-pink.svg)

一個結合「精準貸款理財計算」與「浪漫櫻花美學」的專業網頁工具。
A professional web-based loan calculator merging precise financial logic with romantic Japanese aesthetics.

---

## 🌟 專案特色 | Features

### 看著用著會很讚
- **浪漫美學**：動態櫻花瓣飄落特效，讓試算過程變得優雅輕鬆。
- **人性化輸入**：專為全球華人使用者設計，「輸入 35」即代表「35 萬元」。
- **精準計算**：採用標準「本息平均攤還」公式，確保利息與月付額 100% 準確。
- **一鍵截圖**：內建美學截圖按鍵，自動隱藏 UI 按鈕，生成乾淨的分析報告。
- **移動端優化**：針對 iPhone/Android 瀏覽器排版優化，文字按鈕絕不重疊。

### Highlights
- **Sakura Animation**: Custom keyframe-based falling petal effects for a calming user experience.
- **Precision Math**: Built-in Equal Installment (PMT) algorithm for high-accuracy financial planning.
- **Professional Export**: One-click screenshot feature that auto-hides UI elements for a clean image report.
- **Responsive Design**: Mobile-first approach ensuring perfect layout on all screen sizes.

## 🛠 技術棧 | Tech Stack

- **HTML5 / CSS3**: Custom animations and modern layout.
- **JavaScript**: Pure Vanilla JS logic (No heavy frameworks).
- **html2canvas**: High-fidelity DOM-to-PNG export.
- **Google Fonts**: Libre Baskerville & Inter.

## 🚀 使用說明 | How to Use

1. **下載檔案 (Download)**: 下載 `index.html` 檔案至您的電腦。
2. **運行專案 (Run)**: 使用瀏覽器直接開啟檔案即可。
3. **計算試算 (Calculate)**:
   - 輸入貸款金額（例如 35 代表 35 萬）。
   - 輸入還款期數與年利率。
   - 數據將即時更新。
